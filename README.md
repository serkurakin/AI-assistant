# 🔬 Scientific AI Agent (RAG + Web Search)

Интеллектуальный ассистент для анализа научных статей. Система построена на базе **LangChain Agents**, что позволяет ИИ не просто отвечать на вопросы, а самостоятельно выбирать инструменты для поиска информации.

## Ключевые особенности

- **Agentic Workflow**: Использование `AgentExecutor`.
- **Гибридный поиск**: Агент приоритизирует поиск в локальной базе знаний (RAG), а при отсутствии данных обращается к поиску в реальном времени (Web Search).
- **Метаданные и цитирование**: Система автоматически извлекает названия PDF-файлов и расставляет ссылки вида `[article.pdf]` после каждого утверждения.
- **Очистка**: Скрипт индексации автоматически отсекает разделы "References" в статьях, чтобы база данных содержала только смысловой текст.
- Бот запоминает историю сообщений (до 3 штук).

## Стек технологий

*   **LLM**: GPT-4o-mini (через OpenRouter)
*   **Framework**: LangChain (Core, Classic, Community)
*   **Vector DB**: ChromaDB (Persistent Storage)
*   **Embeddings**: Sentence-Transformers (paraphrase-multilingual-MiniLM-L12-v2)
*   **Search API**: Tavily Search
*   **Telegram API**: pyTelegramBotAPI

## Структура

*   `bot_v3.py` — Основной файл ИИ-агента и Telegram-бота.
*   `rag.py` — Скрипт для первичной обработки PDF и создания векторной базы.
*   `articles/` — Папка для размещения научных публикаций (PDF).
*   `chroma_db/` — Локальное хранилище векторных представлений (создается автоматически).

## Инструкция по запуску

### 1. Подготовка окружения
Установите необходимые зависимости:
```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt

### 2. Подготовка папок и данных
1. Создайте в корне проекта папку `articles/`.
2. Поместите в неё ваши статьи в формате **PDF**.

### 3. Настройка переменных окружения (.env)
Создайте файл `.env` и добавьте в него ваши ключи API:

```env
BOT_TOKEN=ваш_токен_телеграм
OPENROUTER_API_KEY=ваш_ключ_openrouter
TAVILY_API_KEY=ваш_ключ_tavily

### 4. Создание векторной базы знаний (RAG)
Перед запуском бота необходимо индексировать статьи. Запустите скрипт:
```bash
python rag_v3.py

### 5. Запуск ИИ-агента
После того как база данных успешно создана, запустите основного бота:
```bash
python bot_v3.py
